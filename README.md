**Project Overview:**  

LeafLog  is a light-themed, visually soothing web app that helps individuals build sustainable habits through daily eco-friendly missions and personal action journaling. Designed with a soft green palette and a calm, nature-inspired aesthetic, LeafLog turns environmental responsibility into a rewarding daily ritual. Users are encouraged to complete simple predefined tasks while also logging their own actions in a beautifully crafted moodboard-style feed. With gentle animations, streak tracking, and a clear sense of personal impact, LeafLog makes going green feel both meaningful and visually delightful â€” helping users bloom into a better version of themselves, one day at a time.
